# Event Landing Page - React + Vite + Tailwind CSS

A fully customizable, semi-custom landing page template built with React, Vite, and Tailwind CSS. Perfect for creating personalized event landing pages for potential clients.

## Features

- **React 19** - Latest React with modern hooks
- **Vite** - Lightning-fast development server and build tool
- **Tailwind CSS** - Utility-first CSS framework
- **6 Color Themes** - Pre-designed gradient color schemes
- **9 Customizable Sections** - Each with 3 unique layout options
- **Real-time Customization** - Live preview of all changes
- **Persistent Settings** - Saves preferences in localStorage
- **Export/Import** - Save and share customization configurations
- **Responsive Design** - Mobile-first, works on all devices
- **Smooth Animations** - Professional transitions and effects

## Getting Started

### Installation

```bash
# Navigate to the app directory
cd landingPage/app

# Install dependencies (already done)
npm install

# Start development server
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Quick Start

1. **Run the app**: `npm run dev`
2. **Open browser**: Navigate to `http://localhost:5173`
3. **Click "Customize"**: Open the control panel on the right
4. **Try themes**: Select different color themes
5. **Change layouts**: Pick different layouts for each section

## Customizing Content

All event content is in `src/data/eventData.js`. Edit this file to customize:

- Event name, logo, dates, location
- About section content
- Categories and requirements
- Timeline and milestones
- Prizes and benefits
- Jury profiles
- Documentation and winners
- Sponsor logos
- Contact info and FAQ

## Using the Control Panel

- Click "Customize" button or press **Ctrl+K**
- Select color theme
- Choose layouts for each section
- Export/import settings
- Reset to defaults

## Keyboard Shortcuts

- **Ctrl/Cmd + K** - Toggle customization panel

## Deployment

```bash
npm run build
# Deploy the 'dist' folder to your hosting
```

## Tips

1. Start by customizing `src/data/eventData.js`
2. Choose a color theme that matches the event brand
3. Select layouts that best showcase your content
4. Export settings to save your configuration
5. Replace placeholder images with real event photos
